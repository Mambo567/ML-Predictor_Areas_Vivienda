# Modelo Predictor de Áreas Habitacionales

## Descripción del Proyecto
Este proyecto desarrolla un conjunto de modelos predictivos para estimar las áreas de diferentes secciones de una vivienda: dormitorios, baños, salón, cocina y aseos. La herramienta está diseñada para ayudar a empresas, como una de servicios de pintura, a optimizar la planificación de recursos y el presupuesto en función de las dimensiones estimadas de los espacios.

---

## Objetivo
Proporcionar una solución práctica, precisa e interpretable para predecir las áreas habitacionales de una vivienda utilizando variables disponibles al momento de la consulta.

---

## Características Principales
- **Modelos Predictivos**:
  - **Regresión Lineal**: Principal modelo para dormitorios, baños, salón y cocina.
  - **Random Forest**: Utilizado para predecir el área de aseos.
- **Proporciones Promedio**: Estrategia efectiva para segmentar áreas individuales de dormitorios y baños.
- **Explicabilidad**: Importancia de las variables predictoras para facilitar la interpretación.
